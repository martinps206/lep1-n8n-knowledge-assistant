# LEP-1 Knowledge Assistant

## Vision

Construir un asistente de conocimiento personal basado en automatizaciones e inteligencia artificial capaz de capturar, procesar, clasificar y almacenar información proveniente de múltiples fuentes.

## Objectives

* Aprender n8n profesionalmente.
* Aprender integración de IA.
* Aprender automatización empresarial.
* Construir portfolio.
* Crear una base de conocimiento reutilizable.

## Success Criteria

* Capturar información automáticamente.
* Generar resúmenes automáticos.
* Clasificar contenido.
* Almacenar información estructurada.
* Recuperar conocimiento eficientemente.
